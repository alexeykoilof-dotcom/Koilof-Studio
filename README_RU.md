# Lyrics Video Studio Ultra

Локальная студия для lyric-video: выбираешь аудио, вставляешь текст, смотришь превью, настраиваешь текст/фон/фрагмент и собираешь MP4.

## Запуск на Windows

1. Распакуй архив.
2. Открой папку `lyrics-video-studio-ultra`.
3. Запусти:

```bat
START_WINDOWS.cmd
```

Скрипт сам проверит Python, ffmpeg и зависимости. Если Windows попросит разрешение — разреши.

## Как работать

1. Нажми **Выбрать аудио**.
2. Вставь текст в большое поле.
3. Укажи старт и финиш фрагмента песни, например `9` и `30`.
4. Смотри превью: кнопка **Запустить / Пауза** находится под превью.
5. Настраивай рядом с превью: разрешение, FPS, размер текста, цвет, позицию, анимацию текста, движение фона.
6. Нажми **Сгенерировать видео**.
7. Нажми **Открыть папку с готовыми видео**.

## Что изменено в Ultra

- все настройки перенесены рядом с превью;
- кнопка превью теперь одна: запуск и пауза в одной кнопке;
- добавлена громкость превью;
- убрана лишняя заглушка в превью;
- оформлены кнопки выбора файлов;
- убраны лишние кнопки, которые не нужны в рабочем процессе;
- добавлены переходы текста: fade, slide up, slide down, slide left, zoom, pop, glow;
- добавлены движения фона: slow zoom, zoom out, pan left/right/up/down;
- старт/финиш реально обрезают итоговое видео.

## Важно про автотайминг

Программа не распознаёт голос. Она распределяет строки по выбранному фрагменту песни автоматически: равномерно или по длине строки. Если нужна идеальная синхронизация с вокалом, подправь секунды в редакторе тайминга и проверь через превью.

## Где готовое видео

Готовые файлы сохраняются в папку:

