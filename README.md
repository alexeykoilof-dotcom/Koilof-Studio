# 🎬 Koilof Studio

### Локальный видеоредактор для роликов, титров, lyric-video и короткого контента

**Koilof Studio** — это настольная и веб-студия для быстрого монтажа видео: добавляйте аудио, фоновые изображения и ролики, текстовые клипы, субтитры, эффекты, переходы и экспортируйте готовый MP4.

## ✨ Что это такое

Koilof Studio — локальный редактор видео, который запускается на компьютере и открывается как desktop-приложение или как локальная веб-страница в браузере.

Проект рассчитан на создание:

- lyric-video;
- клипов с титрами;
- коротких роликов для TikTok, Reels, Shorts;
- видео с музыкой, текстом и фоном;
- простых рекламных и презентационных роликов;
- контента с вертикальным, горизонтальным или квадратным форматом.

Главная идея проекта — дать понятный монтажный интерфейс без тяжёлых профессиональных программ.

## 🚀 Возможности

### 🎞 Монтаж видео

- Добавление видео, изображений и фоновых материалов.
- Таймлайн с видео, текстом и аудио.
- Разрезание клипов.
- Дублирование, копирование и вставка клипов.
- Перемещение курсора по таймлайну.
- Управление длительностью проекта.

### 🎵 Работа со звуком

- Загрузка аудио: MP3, WAV, M4A и другие поддерживаемые форматы.
- Настройка громкости.
- Обрезка начала и конца трека.
- Fade in / fade out.
- Предпрослушивание звука прямо в редакторе.

### ✍️ Текст и титры

- Добавление текстовых клипов.
- Импорт текста из TXT / SRT / LRC.
- Автоматическое распределение строк по времени.
- Настройка размера, цвета, обводки и выравнивания.
- Несколько стилей текста: Clean, Bold, Poster, Soft.
- Анимации появления: fade, slide, zoom, pop, blur, typewriter, glow.

### 🖼 Фоны и визуальные эффекты

- Встроенная библиотека фоновых пресетов.
- Поиск изображений через Openverse.
- Поддержка фото и видео в качестве фона.
- Размытие фона.
- Движение фона: zoom, pan, static.
- Chroma key.
- Удаление тёмного или светлого фона.

### 🤖 ИИ-помощник для текста

Koilof Studio может подключаться к локальному Ollama и помогать с текстом:

- разбить текст на короткие строки;
- сократить текст;
- исправить русский текст;
- усилить первые строки;
- подготовить текст для титров или lyric-video.

По умолчанию используется локальная модель:

```text
qwen2.5:1.5b
```

ИИ не требует API-ключей, если Ollama запущен локально.

### 📤 Экспорт

- Экспорт в MP4.
- Форматы проекта:
  - 9:16 — TikTok / Reels / Shorts;
  - 16:9 — YouTube;
  - 1:1 — квадратный формат;
  - Custom — свои размеры.
- Настройка FPS: 24, 30, 60.
- Выбор качества: черновик, оптимальное, лучшее.
- Готовые видео сохраняются в папку `output/`.

## ⚙️ Требования

Перед запуском нужны:

- Python 3.10+;
- FFmpeg;
- браузер или desktop-окно через pywebview;
- Ollama — только если нужны ИИ-функции.

На Windows стартовые `.cmd`-файлы сами пытаются проверить Python, FFmpeg и установить зависимости.

---

## ▶️ Запуск на Windows

### Desktop-режим

Запустите файл:

```bat
START_DESKTOP.cmd
```

После запуска откроется окно Koilof Studio.

### Веб-режим в браузере

Запустите файл:

```bat
START_WINDOWS.cmd
```

Приложение откроется локально по адресу вида:

```text
http://127.0.0.1:8765
```

---

## 🐧 Запуск на Ubuntu / Linux

```bash
chmod +x START_UBUNTU.sh
./START_UBUNTU.sh
```

Или вручную:

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
python app.py
```

---

## 🧠 Подключение Ollama

ИИ-функции работают через локальный Ollama.

Установите Ollama и скачайте модель:

```bash
ollama pull qwen2.5:1.5b
```

Запустите Ollama, затем откройте Koilof Studio. Приложение проверит подключение автоматически.

Можно переопределить адрес и модель через переменные окружения:

```bash
OLLAMA_URL=http://127.0.0.1:11434
OLLAMA_MODEL=qwen2.5:1.5b
```

---

## 📦 Сборка EXE на Windows

После установки зависимостей можно попробовать собрать исполняемый файл:

```bat
BUILD_EXE_WINDOWS.cmd
```

Сборка использует PyInstaller.

## 🧪 Как пользоваться

1. Создайте новый проект.
2. Выберите формат: 9:16, 16:9, 1:1 или Custom.
3. Добавьте аудио.
4. Добавьте видео, изображение или встроенный фон.
5. Вставьте текст или импортируйте TXT / SRT / LRC.
6. Настройте таймлайн, стили, анимации и переходы.
7. Нажмите **Экспорт MP4**.
8. Заберите готовый файл из папки `output/`.

## ⌨️ Горячие клавиши

| Клавиша | Действие |
|---|---|
| `Space` | Play / Pause |
| `V` | Курсор |
| `B` | Ножницы |
| `S` | Разрезать выбранный клип |
| `Ctrl + D` | Дублировать |
| `Ctrl + C` | Копировать клип |
| `Ctrl + V` | Вставить на курсор |
| `Delete` | Удалить выбранный клип |
| `ПКМ` | Контекстное меню клипа |

## 📌 Важно

Koilof Studio не распознаёт вокал автоматически. Для lyric-video программа может распределять строки по выбранной длительности, но точную синхронизацию лучше править вручную на таймлайне.

Для экспорта MP4 обязательно нужен FFmpeg.
